# A Story of Big Sean

## Description

We decided to write a story about Big Sean using code. To acomplish this, we made an app starting from Big Seans birth to his success today, and displayed what Big Sean did to give back to the Detroit community. 

### Features

- make Big Sean cry as a baby
- tab view to switch between chapters
- cover photo
- scroll views

### Built with

- XCode
- Swift
- Swift UI
- Sketch
